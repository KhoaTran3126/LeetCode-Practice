# Bit Manipulation
Contain problems relevant to bit manipulation operators (OR, XOR, AND, NOT).

## XOR Operator, given k > 0
1. XOR(k, 0) = k
2. XOR(k, k) = 0

## AND Operator, given k > 0
1. AND(k, 0) = 0
2. AND(k, k) = k

## OR Operator, given k > 0
1. OR(k, 0) = k
2. OR(k, k) = k
3. OR(0, 0) = 0

## NOT operator, given k > 0
1. NOT(1) = 0
2. NOT(0) = 1
3. NOT(k) = 0
